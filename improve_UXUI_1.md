UI/UX Redesign Prompts for VietCinema
Phase 1: Visual Foundation & Brand Refinement
Design a modern, sophisticated cinema booking website interface with the following specifications:

COLOR PALETTE:
- Primary: Deep Gold (#D4A574) for CTAs and accents
- Background: Rich Charcoal (#1A1A1A) for main background
- Cards: Warm Gray (#2D2D2D) with subtle elevation
- Hover/Active: Copper (#B87D4B)
- Text Primary: Soft Cream (#F5F1E8)
- Text Secondary: Muted Slate (#6B6B6B)
- Featured Badge: Cinematic Red (#C85A54)

TYPOGRAPHY:
- Display/Movie Titles: Cinzel (700-800 weight) for premium feel
- Headings: Inter or Archivo (700-800 weight)
- Body Text: Inter (400-500 weight)
- Size Scale: 12px/14px/16px/20px/24px/32px/48px
- Line Height: 1.5 for body, 1.2 for headings

SPACING SYSTEM:
- Base unit: 8px
- Card spacing: 24px gaps between movie cards
- Section padding: 64px vertical, 80px horizontal (desktop)
- Card padding: 24px internal
- Component margins: 16px between related elements

COMPONENT SPECIFICATIONS:

Homepage Hero Section:
- Large movie poster with gradient overlay (bottom to top: rgba(26,26,26,0.9) to transparent)
- Title in Cinzel, 48px
- Rating stars (gold) + score + duration + release date in one line
- Genre tags with subtle borders, 8px border-radius
- Two CTAs: Primary "Đặt Vé Ngay" (gold button) + Secondary "Xem Trailer" (outlined copper)
- Button specs: 16px vertical padding, 32px horizontal, 8px border-radius, Inter 600

Movie Card Design:
- Aspect ratio 2:3 for poster
- 12px border-radius
- Subtle shadow: 0 4px 24px rgba(0,0,0,0.4)
- Hover effect: transform translateY(-4px), shadow increases
- Badge positioning: top-right corner, 8px from edges
- Title below poster: 16px, max 2 lines with ellipsis
- Genre + rating row: 14px, muted slate color
- Spacing between cards: 24px

Primary Button Style:
- Background: Deep Gold (#D4A574)
- Text: Rich Charcoal (#1A1A1A), Inter 600
- Padding: 16px 32px
- Border-radius: 8px
- Hover: Copper (#B87D4B) background
- Active: slightly darker, scale(0.98)
- Add subtle shine gradient overlay

Secondary Button Style:
- Border: 2px solid Copper (#B87D4B)
- Background: transparent
- Text: Soft Cream (#F5F1E8)
- Same padding and radius as primary
- Hover: background Copper with 0.1 opacity

LAYOUT STRUCTURE:
- 12-column grid system
- Max-width: 1440px centered
- Gutters: 24px
- Movie grid: 4 columns desktop, 2 tablet, 1 mobile
- Section spacing: 80px between major sections

ELEVATION/SHADOWS:
- Cards at rest: 0 4px 24px rgba(0,0,0,0.4)
- Cards hover: 0 8px 32px rgba(0,0,0,0.6)
- Modals: 0 24px 48px rgba(0,0,0,0.8)
- Subtle inner shadow on inputs: inset 0 2px 4px rgba(0,0,0,0.3)

VISUAL DETAILS:
- All images: 8-12px border-radius
- Badges: 4px border-radius, 6px padding vertical, 12px horizontal
- Genre tags: outlined style, 1px border, 16px padding horizontal
- Rating stars: 16px size, gold color with slight glow
- Icons: 20-24px, use Lucide or Heroicons set
- Smooth transitions: 200ms ease-in-out for all interactive elements

Create a clean, premium aesthetic that feels like a high-end cinema experience, avoiding harsh neon colors and maintaining sophistication throughout.
